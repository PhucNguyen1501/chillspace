# Extension Icons

Place the following icon files in this directory:

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Icon Requirements

All icons should:
- Be PNG format
- Have transparent backgrounds
- Represent the extension's purpose (API documentation/queries)
- Follow Chrome Web Store icon guidelines

## Generating Icons

You can use tools like:
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [Canva](https://www.canva.com/)
- Adobe Illustrator/Photoshop
- Figma

## Placeholder Icons

For development, you can use simple colored squares or the default Chrome extension icon until proper branding is ready.
